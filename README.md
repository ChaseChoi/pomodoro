# pomodoro
A pomodoro app to perform time management

## Podfile
install cocoapods: 
1. `sudo gem install cocoapods`
2. `pod update`
```
target 'pomodoro' do
  # Comment the next line if you're not using Swift and don't want to use dynamic frameworks
  use_frameworks!

  # Pods for pomodoro
    pod 'Charts'
end
```
